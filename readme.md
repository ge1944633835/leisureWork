## 主要用的技术点有：

1. 项目整体的架构设计采用的是mvp设计模式，分为4个模块，分别为新闻、video、天气、说明

2.  设计风格采用的是Google Material Design 设计风格

3. 在网络请求与处理的地方采用的是Retrofit+Rxjava

4. 在video模块采用的是jcvideoolayer开源库，也涉及到shareSdk社会化分享等等
